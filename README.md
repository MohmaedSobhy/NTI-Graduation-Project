# 🔐 Secure Network Infrastructure Project

> A comprehensive enterprise-grade network security implementation built in **Cisco Packet Tracer**, covering end-to-end network design with modern security protocols and monitoring.

---

## 📋 Overview

This project demonstrates the design and implementation of a fully secured enterprise network environment. It integrates dual-stack IPv4/IPv6 addressing, layered security controls, segmented traffic zones, and centralized monitoring — simulating real-world network infrastructure for a secure organization.

---

## 🚀 Features & Implementation

### 🌐 Network Design & Addressing
- Designed and implemented a secure network topology using both **IPv4** and **IPv6** addressing schemes
- Structured internal, external, and DMZ zones for controlled and logical traffic flow

### 🛡️ Layer 2 Security
- Configured **VLANs** and **native VLANs** to segment network traffic and reduce broadcast domains
- Applied **DHCP** for dynamic IP address assignment across network segments
- Enabled **DHCP Snooping** to prevent rogue DHCP server attacks and protect Layer 2 integrity

### 🔒 Access Control & Firewall
- Implemented **Access Control Lists (ACLs)** to filter and control inbound and outbound traffic
- Configured **zone-based security policies** to enforce traffic rules between network zones
- Set up a **stateful firewall** to inspect and manage traffic across internal, external, and DMZ boundaries

### 🔑 VPN & Routing
- Established a **VPN (Virtual Private Network)** for secure encrypted remote access
- Configured **OSPF (Open Shortest Path First)** as the dynamic routing protocol for efficient and scalable routing within the network

### 🏢 Zone Management & DMZ
- Managed **internal and external zones** with strict policy enforcement
- Controlled access to **DMZ-hosted services** (e.g., web, DNS) while isolating them from the internal network

### 📡 Network Monitoring & Time Synchronization
- Configured an **NTP (Network Time Protocol) server** to ensure accurate and synchronized timestamps across all devices
- Set up a **Syslog server** to centralize log collection for real-time network monitoring, auditing, and incident response
